# PSWE-01 SPA

## Levantar el proyecto

Ejecuta estos comandos desde la raíz del proyecto: `PSWE-01/`

1. Instala dependencias:

```bash
npm install
```

2. Configura variables de entorno:

```bash
cp .env.example .env
```

3. Inicia el servidor de desarrollo:

```bash
npm run dev
```

La app correrá en la URL que indique Vite en consola, normalmente `http://localhost:5173`.
